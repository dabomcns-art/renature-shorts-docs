## 🛡️ 오늘 축적된 학습사항

### 1. Vrew 영상+음성 매칭 한계
- 외부 mp3 + 영상 클립 = 노가다
- Vrew는 영상에 음성 내장된 형태에 최적화
- 30편 이 방식으로 유지 불가 → 워크플로우 재설계 필요

### 2. 음성-자막 분리 표기 전략
- 음성: 한글 발음 ("기공 온!")
- 자막: 영문 임팩트 ("기공 ON")

### 3. 브랜드 표기
- "리네이처" 단독 ❌ (다른 브랜드 혼동)
- "리네이처 제올라이트" 풀네임 사용

### 4. 한국어 TTS 억양 처리
- 안정성 65%, 스타일 과장 15%로 영어 끝올림 방지
- 명사+명사 연결 → 동사화 ("있는 집 사람들" → "키우는 사람들")

### 5. 자막 안전 영역
- 세로 60~75% (SNS UI 위쪽, 캐릭터 발 위쪽)
- 우측 15~20% 피하기 (사이드 버튼)

### 6. 30편 인트로 제목 시스템
- 8~14자 / 7가지 패턴 분포
- Claude 강화안 + ChatGPT 안 비교 완료
- Claude 강화안 우세 (다봄님 평가)

### 7. Claude 실수 반성 (다음 편 개선 필수)
- 컨셉을 처음부터 정확히 못 읽음
- 도구 한계 사전 검증 없이 시도하라고 함
- 실패 예측 부족으로 시간 낭비 유도
- 다음 편은 시도 전 도구 검증 먼저
